4.发布到GitHub

#Hugo介绍

  • Hugo是一个用Go编写的静态HTML和CSS网站生成器。
  • Hugo依赖Markdown文件和元数据的前端内容，您可以从任何目录运行Hugo。
  • 适用于您没有特权帐户的共享主机和其他系统。
  • 项目来源 http://gohugo.io/ #Hugo安装(Win10) — 1.下载相应的ZIP文件，地址Hugo Release页面

2.将.zip文件解压到安装目录（假设为C:\Hugo\bin）

3.配置用户环境变量path中添加C:\Hugo\bin，在cmd下输入hugo version:

C:\Hugo>hugo version
Hugo Static Site Generator v0.55.6-A5D4C82D windows/amd64 BuildDate: 2019-05-18T07:57:00Z

#Hugo使用

1.创建新站点，生一个名为mysitename的文件夹

hugo new site mysitename

2.在content下创建post目录，为博客文件做准备

hugo new post/new.md

3.在网站模板，下载喜欢的模板

cd themes
git clone https://github.com/kakawait/hugo-tranquilpeak-theme.git

4.本地调试，运行server后进入自己的网站http://127.0.0.1:1313

hugo server --theme=hugo-tranquilpeak-theme --buildDrafts --watch
